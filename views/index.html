Welcome to the X4 Analyzer

This is the compiled web interface for the X4 Analyzer. The program is used to extract and process useful data from a X4 game save file. That file consists of more than 6 million lines of XML. The 'compiling' of the data is done with a python script that stores the useful data in a SQLite database. That database can then be used by the Go web server to display said data.
